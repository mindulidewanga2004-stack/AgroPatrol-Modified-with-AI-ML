The system uses computer vision models to detect and identify different types of animals in real time from camera input captured in agricultural fields.
The detected animal type and related metadata are stored in a real-time database, and alerts are instantly delivered to farmers through a web application, enabling quick and informed responses to prevent crop damage. This solution transforms traditional farm monitoring into a data-driven, intelligent decision-making system.
The project involved image preprocessing, model training and evaluation, real-time inference, database integration, and web-based alerting, providing hands-on experience in building an end-to-end AI-powered IoT solution for real-world agricultural challenges.
